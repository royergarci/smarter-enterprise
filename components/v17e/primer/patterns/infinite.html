Infinite Scroll

What it is

A feature that pre-fetches content from a subsequent page and dynamically adds it to the current page.

Business Impact

Allows users to scroll through content on a single page without clicking a button to navigate to a separate page.

When

To be used on a page where all content does not fit on initial load and when a user is unlikely to navigate away to view more content.

Basic Functionality

As a user scrolls down a page, a ‘Loading More’ notification appears, and subsequently loads more content onto the current page. As a user scrolls to the bottom of the page, filters lock at the top of the screen to allow users to manipulate filters without having to scroll to the top of the page. The ‘Loading More’ notification will disappear when all content has loaded.
