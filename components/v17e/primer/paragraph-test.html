Col 1-1

Column size variations

Large

Apply "large" class to ibm-col-1-1, or any other column variation, to change the font size of all text contained in the column. This applies to all headers and paragraphs.

			<div class="ibm-columns">
			<div class="ibm-col-1-1 large">

Default

Do not apply an additional class to ibm-col-1-1, or any other column variation, to keep the default font size of all text contained in the column. This retains font size of all headers and paragraphs.

			<div class="ibm-columns">
			<div class="ibm-col-1-1">

Small

Apply "small" class to ibm-col-1-1, or any other column variation, to change the font size of all text contained in the column. This applies to all headers and paragraphs.

			<div class="ibm-columns">
			<div class="ibm-col-1-1 small">
