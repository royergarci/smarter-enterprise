This is a popup page sample. The HTML code of the page is exactly the same as any other templates with the following differences:

  • there is a .ibm-popup class on the #ibm-top element:
    <div id="ibm-top" class="ibm-popup">
    The dynamicnav is automatically disabled if this code is detected.
  • there is a new #ibm-title element below the #ibm-unav-links containing the page title:
    <li id="ibm-title">Text chat</li>
  • in case you cannot modify the masthead to add a title you can also add this code to the <head> section of the page:
    <script type="text/javascript">
    ibmweb.config.set({pagetitle: "Some page title"});
    </script>
